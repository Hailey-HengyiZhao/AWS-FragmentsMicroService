## Introduction

## Run command

To run the server
```
npm start
```

To run the server with nodemon
```
npm run dev
```

To run all unit tests
```
npn run test
```

To run all hurl integration tests
```
npm run test:integration
```

To connect to debugger and run the server
```
npm run debug
```

To run ESLint
```
npm run lint
```

To test the coverage of Unit testing
```
npm run coverage
```
